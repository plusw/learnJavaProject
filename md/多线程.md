## java多线程
并发:在同一时刻,有多个指令在cpu上交替执行  
并行:在同一时刻，有多个指令在cpu上同时执行  
#### 多线程的创建
#####继承Thread来创建多线程
创建线程类  
```java
public class Mythread extends thread{    
@Override  
public void run(){//重写run方法    
  for(int i=0;i<100:i++){   
      System.out.println(i);  
  }  
}  
```
建立测试类  
```java
public class Demo{  
  public static void main(String[] args){  
    MyThread t1=new MyThread();  
    t1.start();  
  }    
}  
```
